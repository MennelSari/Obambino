ABSENCE :


Recuperer toute les absence [get] 
=> /api/absence/list


Recuperer une absence [get]
=> /api/absence/show/{id}


Créer une absence [POST]
=> /api/absence/create


Modifier une absence [PUT]
=> /api/absence/update/{id}


Supprimer une absence [delete]
=> /api/absence/delete/{id}
